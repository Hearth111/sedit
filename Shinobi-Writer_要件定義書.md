# 開発依頼：TRPGシナリオエディタ (.exe配布用)

## 技術スタック
* **言語:** Python 3.x
* **GUIライブラリ:** Flet (モダンなUIを簡単に作るため)
* **PDF生成:** Typst (外部コマンドとして呼び出し) または ReportLab
* **配布:** PyInstaller で .exe 化する前提

## アプリケーションの構成
1. **画面レイアウト**
   * ウィンドウ全体をダークテーマに設定。
   * **左側:** テキスト入力エリア (Multiline TextField)。
   * **右側:** 設定パネル (タイトル、画像パス、出力ボタン)。
   * リアルタイムプレビューは不要（処理を軽くするため）。

## 機能要件
1. **編集機能**
   * マークダウンライクな記法 (`#`, `>`, `{{HO}}`) を入力可能にする。
2. **PDF出力機能 (最重要)**
   * 「PDF出力」ボタンを押すと、入力されたテキストを解析する。
   * **Typst形式**のファイル (.typ) を一時的に生成する。
   * Typstの機能 (`columns(2)`) を使って、完璧なA4 2段組レイアウトにする。
   * 生成されたPDFをユーザーが指定した場所に保存する。

## デザインの雰囲気
* 配色は「黒(#111)」ベース、「赤(#d00)」アクセント。
* Fletのテーマ機能を使い、スタイリッシュな見た目にする。
